# Synthetic lab

The default lab contains synthetic application classes: immersive XR, industrial automation,
autonomous systems, AI-enabled devices, and emergency low-latency traffic.

Generated traces include:

- traffic demand
- RF interference
- OWC blockage probability
- line-of-sight score
- beam-alignment error
- mobility index
- photonic-efficiency proxy

No real operator data or infrastructure telemetry is required.
